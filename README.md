# L2Physique_LaTeX
Répertoire de prises de notes et de comptes-rendus de travaux pratiques de physique (Licence 2), rédigés en LaTeX.


Bien que je m’efforce de produire des contenus rigoureux et de corriger les erreurs signalées, il est possible que certaines **coquilles, imprécisions ou erreurs de calcul** subsistent. Ces documents ne constituent donc **pas une référence officielle** et doivent être utilisés **avec discernement**.

L’utilisation de ces notes se fait **à vos risques et périls**. Il est fortement recommandé de les confronter aux supports de cours, ouvrages de référence et indications fournies par les enseignants.

Toute remarque ou correction constructive est la bienvenue.
